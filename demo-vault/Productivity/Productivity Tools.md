# Productivity Tools

The current stack. Will be different in a year. Has been different every year.

## Notes

- **NoteForge** for personal notes, journals, knowledge graph. Plain markdown in a folder I can grep.
- **A paper notebook** for ad-hoc capture during meetings. Transcribed weekly into NoteForge or thrown away.

## Tasks

- **Things 3** for personal tasks. Worth the price for the inbox-and-defer flow alone.
- **The work tracker** for work tasks. Don't mix work and personal task systems. They have different rhythms.

## Calendar

- One calendar app for everything: work, personal, recurring habits. Color-coded.
- Block focus time as actual events. If it's not on the calendar it doesn't happen.

## Email

- Three folders, processed daily. See [[Inbox Zero]] for the procedure.

## Time Tracking

Stopped doing it. The data was interesting; the friction wasn't worth it. I've replaced it with the [[Weekly Review Template]] retro question — "where did the week actually go?"

## Files

- iCloud for "personal life" files.
- The work cloud drive for "work" files.
- A local Documents folder for in-progress stuff that doesn't need syncing.
- The desktop is treated as a one-week buffer, not as storage.

## What I've Stopped Using

- **Notion.** Beautiful, slow, and the lock-in finally bothered me.
- **Roam Research.** The graph view was the hook; daily friction was high.
- **Obsidian plugins.** Endless tweaking for marginal gains. I went back to plain markdown and was happier.
- **Multiple to-do apps.** I had three at once for two years. None of them worked because none of them was complete.
