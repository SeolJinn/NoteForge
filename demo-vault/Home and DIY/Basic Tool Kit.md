# Basic Tool Kit

The small set of tools that handle ninety percent of what comes up around the apartment.

## What lives in my box

I keep it deliberately small so I actually know where everything is.

- A claw hammer, nothing fancy
- A screwdriver with swappable bits, flat and Phillips
- An adjustable wrench and a small set of pliers
- A tape measure and a spirit level
- A utility knife with spare blades

If I need more than this, it's usually a job worth renting a tool for.

## The extras worth having

These earned their spot after I kept wishing I had them.

- A cordless drill, the single most-used thing I own
- A stud finder so I stop guessing where to hang heavy things
- Painter's tape, it solves more problems than it has any right to
- A headlamp for working in cupboards and under sinks

## How I keep it usable

A tool kit only helps if I can find the tool in the moment.

- Everything goes back in the same spot after every job
- I keep a few sizes of screws, anchors, and nails in a little tray
- Dead batteries get charged the day they die, not the day I need them

## Things I learned

- Buy decent versions of the tools you use often, cheap ones frustrate
- A level genuinely changes how shelves look on the wall
- Measure twice is a cliche because it keeps being true
